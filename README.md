# ssm_demo_coffee

## 简介：
一个简单的ssm框架、maven、bootstrap集成项目。写了一些示例代码。<br>
所有的图片资源是nginx代理的，也将请求分发了一些地址（当然，这些都是这个工程里看不到的）。
